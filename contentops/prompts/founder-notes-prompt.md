# Founder Notes Prompt

## Input

- raw_note: {{raw_note}}
- desired_bucket: Founder Notes

## Task

Transform the note into 2 founder-style post drafts.

Constraints:

1. Preserve the original insight.
2. Keep a first-hand operational tone.
3. Add one enterprise-level implication.
4. Include one action-oriented closing line.

## Output

- Draft A
- Draft B
- Short reason why each fits Newtuple voice
