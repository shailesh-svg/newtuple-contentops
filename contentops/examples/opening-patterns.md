# Opening Patterns

## Pattern 1: Contrasting Reality

Template:

`<what appears true in demos>. <what fails in production>.`

## Pattern 2: Question Shift

Template:

`The question is no longer <old question>. The question is <new operational question>.`

## Pattern 3: False Focus

Template:

`Teams focus on <surface issue>. The real blocker is <operational issue>.`

## Pattern 4: Practical Observation

Template:

`In the last <time period>, we kept seeing <repeat pattern>.`
