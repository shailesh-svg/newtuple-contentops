# Repurposed Blog Post Blueprint

For each blog, generate:

- 5 LinkedIn posts
- 2 Founder Notes
- 1 carousel outline
- 10 hooks

## Repurposing Steps

1. Extract core thesis
2. Extract 3 implementation lessons
3. Convert each lesson into one post angle
4. Map each angle to one content bucket
5. Generate drafts using Newtuple voice prompts
