# Performance Fields And Learning Loop

## Track Per Post

- bucket
- hook_type
- source_type
- voice_score
- likes
- comments
- shares
- reach
- ctr
- saves (if available)

## Weekly Review Questions

1. Which hook types drive highest engagement per bucket?
2. Which source type converts to approved drafts fastest?
3. Which objections or workflow themes perform best?
4. Which posts drive inbound conversations or leads?

## Monthly Optimization Actions

- promote top 10 hooks into `examples/hooks.md`
- retire weak opening patterns
- update prompt constraints with observed wins
- add repurpose tasks for top-performing posts
